Front End Languages

These languages are the pure fundamentals of any website and they comprise the front-end of any web application.

HTML5
Plain user interface.

HTML is the forefront of the front-end. It communicates with the browser, telling it what to display. It may also link to other sites for assets like Style Sheets and Scripts. HTML uses tags (known as elements) and DOM, which is very simplistic.
I am very advanced in this language and it is the first front-end language that I ever learnt.

My YouTube Video Find out more!
CSS3
Styling for plain documents.

A CSS file is used in order to describe the presentation of a HTML document. This allows for complete control over how the website looks to the end user.
Along with HTML, I am very experienced in CSS and it was one of the first front-end languages that I learnt.

Find out more!
Bootstrap
User Interface Framework

Bootstrap uses more advanced CSS tricks in order to allow for appropriate scaling across all devices. It also has different presets and templates to allow for an easier and quicker experience for a developer. This site actually uses Bootstrap!
I have very vast experience with Bootstrap and I'm very fluent and confident in it.

Find out more!
JavaScript
Front-end code execution.

JavaScript is used for User Experience and controls the HTML document. It controls how elements act and respond to different types of input.
JavaScript is one of my favourite languages to use. It may not be high level but it is extremely flexible and can be used in many different ways.
