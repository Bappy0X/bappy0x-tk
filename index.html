Where Should I Start Coding?

There are many different routes and pathways to coding, and there are so many different ways to start too. Here are a small list of some of the ways you can start, ordered from easiest, to hardest:

  • Rabbids Coding

    Rabbids Coding is all about learning through play. You can place blocks of code down, in order to instruct `Rabbids` on where to go, what to do. This will introduce you to some basic fundamentals within coding. Including loops, conditions and optimisation.

  • Scratch

    Scratch is a lot like Rabbids Coding, however it will introduce you to a lot more, and in a slightly more advanced way. Scratch uses `drag and drop` blocks of code. It can in order to create mini-games and mini applications Scratch can also run in the browser, so no downloads are needed at all. For those who already know languages like JavaScript, custom code blocks can be made - there are some interesting videos online demonstrating this.

  • Roblox

    Roblox is a great starting place for anybody that is interested in multiplayer interaction. Roblox is not as a simple as Rabbids Coding, nor Scratch. However, its unique take on LUA helps for a simple and intuitive start. Roblox is also very well documented, allowing you to search up any issues that you may be having. Roblox's games are all built by their players, this allows anybody to play and create their own custom worlds and there are very many different genres of games to play. It's certainly a great starting point - but I recommend having atleast a little bit of prior knowledge if you choose to code on it.

  • Python

    Python is also one of the greatest places to start. It introduces proper programming syntax and other fundamental/core structures for nearly every language that exists. It also allows its users to use packages that other users built. This allows for an absolutely gigantic amount of functionality, allowing users to do do (quite literally) whatever they'd like to do. This could be; build a game, create a 3d engine, analyse and scan images and infinitely more.
